# Fei Protocol Brand Kit

This is the Fei Protocol Brand Kit. We have two tokens:

`$FEI` - Fei USD

`$TRIBE` - Tribe

We also provide liquidity to exchanges with the `$FEI` - `$TRIBE` pair.
